AQHI AI FORECAST MODEL
Python TensorFlow/Keras scikit-learn Pandas

AQHI Prediction Using LSTM Neural Network

My Role: AI / Machine Learning Developer

An AI project that predicts the Air Quality Health Index (AQHI) from historical air quality data. I used Python to clean the dataset, generate AQHI values from multiple pollutant indicators, and train a stacked LSTM model for time-series forecasting.

  • Preprocessed air quality data by converting formats, handling missing values, and selecting numeric features
  • Calculated AQHI from multiple pollutant measurements and normalized the sequence data for training
  • Built a four-layer LSTM network with dropout to predict future AQHI values from 60-step sequences
  • Evaluated model performance using RMSE, MAE, classification accuracy, and a confusion matrix
